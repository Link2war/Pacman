Ce repos possède deux versions de Pacman différentes :
- Version C avec SDL2
- Version C++ avec Bbop-2D

La version C et C++ ont été faites sur Linux (Debian).
- La version C est compatible avec Windows mais la compilation est à faire soi-même.
- La version C++ n'est pas compatible avec Windows.

La méthode de compilation (Linux) est fournie dans le README.md des deux versions.

Les deux versions ne sont pas encore terminée.
